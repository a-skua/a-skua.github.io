## マルチタスク

ついに、という感じ\
マルチタスクは16日目でも扱うらしい\
２つの関数を用意して、それを0.02秒毎に切り替えて実行する処理を作成

```c
void task1 (void)
{
    // foo
    for (;;) {
        // bar
    }
}

void task2 (void)
{
    // foo
    for (;;) {
        // bar
    }
}
```

この２つを切り替えて実行する

## 今までと何が違うの?
